## 🧠 Project Overview

“Rise of Artificial Intelligence” is an interactive storytelling website that explores the evolution of technology from simple tools to intelligent machines. Instead of a traditional static website, this project delivers an immersive experience using scroll-based animations, interactive UI elements, and dynamic transitions.

The story unfolds across five sections: the origin of tools, the birth of computers, the rise of modern AI, the ethical conflict between humans and machines, and a final conclusion emphasizing human responsibility. Each section is carefully designed to visually and emotionally engage the user through animation and interaction.

---

## 🎯 Features

* 🧩 5-section storytelling structure
* 🎢 Scroll-based animations (parallax, reveal effects)
* 🖱️ Interactive elements (hover cards, clickable choices)
* 🎬 Smooth transitions and loading animation
* 📱 Fully responsive design (mobile, tablet, desktop)

---

## ⚙️ Technologies Used

* HTML, CSS, JavaScript
* React (optional)
* GSAP (ScrollTrigger animations)
* Framer Motion / CSS animations

---

## 🎨 Design Approach

The design follows a cinematic storytelling style inspired by modern interactive websites. Dark-to-light transitions represent the evolution of intelligence, while animations guide the user through the narrative journey.

---

## 💡 Story Sections

1. **Hero** – Beginning of human innovation
2. **Evolution** – Rise of computers and logic
3. **Present** – AI in everyday life
4. **Conflict** – Ethical questions about AI
5. **Conclusion** – The future depends on human choices

---
